Registro


Es conveniente en este tipo de aplicaciones ir registrando los eventos que se van produciendo para:

  • depuración
  • registro de errores
  • uso de la aplicación
  • alertas

Introduction to logging in django

Django Logging


Djanjo tiene un modulo para esto, basado en el que hay integrado en   python,   Django Logging



Lo cuentan en:   Django Logging set up

Ejemplo

En settings.py 


				LOG_FILE = 'mi_archivo_de.log'

				LOGGING = {
				    'version': 1,

				    'disable_existing_loggers': False,

				    'formatters': {

				        'verbose': {
				            'format' : "[%(asctime)s] %(levelname)s [%(name)s:%(lineno)s] %(message)s",
				            'datefmt' : "%d/%b/%Y %H:%M:%S"
				        },

				        'simple': {
				            'format': '%(levelname)s [%(name)s:%(lineno)s] %(message)s'
				        },
				    },

				    'handlers': {

				        'file': {
				            'level': 'INFO',
				            'class': 'logging.FileHandler',
				            'filename': os.path.join(BASE_DIR, LOG_FILE),
				            'formatter': 'verbose',
				            'mode':'w'
				        },

				        'console': {
				            'level': 'DEBUG',
				            'class': 'logging.StreamHandler',
				            'formatter': 'simple'
				        }
				    },

				    'loggers': {
				        'django': {
				            'handlers':['file'],
				            'propagate': True,
				            'level':'ERROR',
				        },

				        'resta': {
				            'handlers': ['file', 'console'],
				            'level': 'DEBUG',
				        },
				    }
				}

Ejemplo

En el código 


						import logging
						logger = logging.getLogger('En esta parte del código')

						...

						logger.debug('esto y lo otro')

						....

						logger.info ('ha pasado esto')
						...

						try:
						    ...
						except ValueError as error:
							logger.error(error)
